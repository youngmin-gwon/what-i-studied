## Description

객체의 생성에 관련된 패턴. 객체의 생성절차를 추상화 하는 패턴

- 시스템이 어떤 구체적인 클래스를 사용하는 지에 대한 정보를 캡슐화 한다.
- 이들 클래스의 인스턴스들이 어떻게 만들고 어떻게 서로 맞붙는지에 대한 부분을 완전히 가린다.

## Type

1. [Singleton Pattern (싱글턴 패턴)](Singleton%20Pattern.md): 한 클래스에 한 객체만 존재하도록 제한한다.
2. [Prototype Pattern (프로토타입 패턴)](Prototype%20Pattern.md): 기존 객체를 복제함으로써 객체를 생성한다.
3. [Builder Pattern (빌더 패턴)](Builder%20Pattern.md): 생성 (construction) 과 표기 (representation) 를 분리해 복잡한 객체를 생성한다.
4. [Factory Method Pattern (팩토리 메서드 패턴)](Factory%20Method%20Pattern.md): 생성할 객체의 클래스를 국한하지 않고 객체를 생성한다.
5. [Abstract Factory Pattern (추상 팩토리 패턴)](Abstract%20Factory%20Pattern.md): 동일한 주제의 다른 팩토리를 묶어 줌.
