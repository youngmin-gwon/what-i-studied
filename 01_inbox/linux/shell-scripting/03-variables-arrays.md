## 03. 변수, 문자열, 배열

변수의 스코프, 다양한 속성 부여 방법, 그리고 Bash 의 강력한 데이터 구조인 배열을 다룹니다.

### 1. 변수 속성 및 선언 (declare)

|     옵션      | 설명                        | 특징                     |
| :---------: | :------------------------ | :--------------------- |
|    `-i`     | 정수용 변수 (Integer)          | 산술 평가 후 저장             |
|    `-r`     | 읽기 전용 (Read-only)         | `readonly` 와 동일, 해제 불가 |
|    `-x`     | 환경 변수로 내보내기 (Export)      | `export` 와 동일          |
|    `-a`     | 인덱스 배열 (Indexed Array)    | 일반 배열                  |
|    `-A`     | 연관 배열 (Associative Array) | 키 - 값 쌍 (Bash 4.0+)    |
| `-l` / `-u` | 소문자 / 대문자 자동 변환           | 저장 시 자동 변환             |

### 2. 주요 특수 변수 (Special Variables)

|      변수      | 의미                         | 용도                               |
| :----------: | :------------------------- | :------------------------------- |
|   **`$?`**   | Last Exit Status           | 직전 명령의 성공 여부 (0: 성공)             |
|   **`$0`**   | Script Name                | 현재 실행 중인 스크립트의 파일명               |
| **`$1 ~$9`** | Position Arguments         | 스크립트로 전달된 인자값들 (인덱스)             |
|   **`$#`**   | Argument Count             | 전달된 인자의 총 개수                     |
|   **`$@`**   | All Arguments (Individual) | 모든 인자를 각각의 단어로 처리 (`"$1" "$2"…`) |
|   **`$*`**   | All Arguments (String)     | 모든 인자를 하나의 문자열로 처리 (IFS 구분)      |
|   **`$$`**   | Process ID (PID)           | 현재 셸의 프로세스 ID                    |
|   **`$!`**   | Background PID             | 직전에 실행된 백그라운드 작업의 PID            |

### 3. 배열 및 연관 배열 (Arrays)

| 기능           | 인덱스 배열 (`declare -a`) | 연관 배열 (`declare -A`)        |
| :----------- | :-------------------- | :-------------------------- |
| **초기화**      | `arr=(v1 v2 v3)`      | `arr=([key1]=v1 [key2]=v2)` |
| **값 접근**     | `${arr[0]}`           | `${arr[key1]}`              |
| **모든 값**     | `"${arr[@]}"`         | `"${arr[@]}"`               |
| **모든 키/인덱스** | `"${!arr[@]}"`        | `"${!arr[@]}"`              |
| **배열 크기**    | `${#arr[@]}`          | `${#arr[@]}`                |

### 4. 문자열 연산 및 변환

| 표현식                   | 의미                 | 예시 (var="Hello")    |
| :-------------------- | :----------------- | :------------------ |
| `${#var}`             | 문자열 길이             | `5`                 |
| `${var:offset:len}`   | 부분 문자열 (Substring) | `${var:1:2}` → `el` |
| `${var^^}`            | 전체 대문자 변환          | `HELLO` (Bash 4)    |
| `${var,,}`            | 전체 소문자 변환          | `hello` (Bash 4)    |
| `${var^}` / `${var,}` | 첫 글자 대문자 / 소문자 변환  | `Hello` / `hello`   |

### 5. 사용자 입력 받기 (read)

|  옵션  | 설명                | 특징                      |
| :--: | :---------------- | :---------------------- |
| `-p` | 프롬프트 메시지 출력       | `read -p "Enter: " VAL` |
| `-s` | 입력 내용 숨김 (Silent) | 비밀번호 입력 시 유용            |
| `-a` | 배열 형태로 입력 받음      | `read -a ARR`           |
| `-t` | 타임아웃(초) 설정        | `read -t 5 VAL`         |
| `-r` | 백슬래시 이스케이프 무시     | 원시 문자열 읽기               |
