## 07. 텍스트 처리 툴킷 (grep, sed, awk, coreutils)

리눅스의 강력한 텍스트 처리 도구들을 활용하여 데이터를 검색, 변형, 분석하는 방법을 정리합니다.

### 1. grep (검색)

|  옵션  | 설명                         | 특징                 |
| :--: | :------------------------- | :----------------- |
| `-E` | Extended Regex (ERE) 사용    | `egrep` 과 동일       |
| `-P` | Perl Regex (PCRE) 사용       | 강력한 정규식 엔진         |
| `-i` | 대소문자 무시 (Ignore Case)      |                    |
| `-v` | 패턴 불일치 행 출력 (Invert Match) | 제외 검색              |
| `-r` | 디렉터리 재귀 검색 (Recursive)     |                    |
| `-n` | 행 번호 함께 출력 (Line Number)   |                    |
| `-o` | 매칭된 문자만 출력 (Only Matching) |                    |
| `-F` | 고정 문자열 검색 (Fixed String)   | 정규식 해석 안 함 (속도 빠름) |

### 2. sed (스트림 편집기)

| 명령 / 옵션       | 설명                  | 예시                         |
| :------------ | :------------------ | :------------------------- |
| `s/old/new/g` | 문자열 치환              | `sed 's/foo/bar/g' file`   |
| `d`           | 특정 행 삭제             | `sed '/^#/d' file` (주석 제거) |
| `p`           | 특정 행 출력 (`-n` 과 권장) | `sed -n '1,5p' file`       |
| `-i`          | 파일 직접 수정 (In-place) | `sed -i 's/a/b/' file`     |
| `-n`          | 자동 출력 억제            | 원하는 결과만 `p` 로 출력 시 사용      |

### 3. awk (필드 기반 처리)

|  내장 변수 / 구문   | 설명                            | 특징                  |
| :-----------: | :---------------------------- | :------------------ |
|     `-F`      | 필드 구분자 지정                     | 예: `-F:` (콜론 기준 분리) |
|  `$1, $2, …`  | 각 필드 데이터                      | `$0` 은 행 전체         |
|     `NR`      | 현재 행 번호 (Number of Records)   |                     |
|     `NF`      | 현재 행의 필드 수 (Number of Fields) |                     |
| `BEGIN { … }` | 처리 시작 전 실행 블록                 | 변수 초기화, 헤더 출력       |
|  `END { … }`  | 모든 처리 완료 후 실행 블록              | 합계 출력, 결과 요약        |

### 4. 핵심 Coreutils (데이터 가공)

| 도구 (Tool) | 주 용도     | 주요 옵션                                      |
| :-------- | :------- | :----------------------------------------- |
| **sort**  | 데이터 정렬   | `-n` (숫자), `-r` (역순), `-t` (구분자), `-k` (키) |
| **uniq**  | 중복 제거    | `-c` (카운트), `-d` (중복만), `-u` (유일값만)        |
| **cut**   | 필드 추출    | `-d` (구분자), `-f` (필드 번호)                   |
| **tr**    | 문자 변환/삭제 | `-d` (삭제), `[:lower:]` → `[:upper:]`       |
| **wc**    | 단어/행 카운트 | `-l` (행 수), `-c` (바이트), `-m` (문자)          |
| **xargs** | 명령 인자 전달 | `-0` (NUL 안전), `-n` (인자 수 제한)              |
