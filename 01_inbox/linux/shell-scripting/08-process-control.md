# 08. 프로세스, 잡 제어, 시그널

멀티태스킹 환경에서 프로세스를 관리하고 시그널을 통해 흐름을 제어하는 방법을 다룹니다.

## 1. 잡 제어 (Job Control)
| 명령어 | 설명 | 특징 |
| :--- | :--- | :--- |
| `cmd &` | 백그라운드에서 실행 | 터미널 점유 안 함 |
| `jobs` | 현재 셸의 잡 목록 확인 | `-l` 옵션으로 PID 표시 |
| `fg %n` | n번 잡을 포그라운드로 전환 | |
| `bg %n` | 일지 정지된 n번 잡을 백그라운드 재개 | |
| `disown %n` | 현재 셸과의 연결 해제 (SIGHUP 무시) | |

## 2. 주요 시그널 (Common Signals)
| 번호 | 시그널 | 설명 | 단축키 |
| :---: | :--- | :--- | :---: |
| **1** | **SIGHUP** | 터미널 종료 시 전송 (Hang-up) | |
| **2** | **SIGINT** | 인터럽트 (Interrupt) | `Ctrl+C` |
| **9** | **SIGKILL** | 강제 종료 (무시 불가) | |
| **15** | **SIGTERM** | 일반 종료 요청 (Term) | |
| **19** | **SIGSTOP** | 일시 정지 (Stop, 무시 불가) | |
| **20** | **SIGTSTP** | 터미널 정지 (Stop from TTY) | `Ctrl+Z` |

## 3. 트랩 (trap)
스크립트 내에서 특정 이벤트(시그널 등)가 발생했을 때 수행할 동작을 정의합니다.

| 이벤트 (Signal/Event) | 설명 | 특징 |
| :--- | :--- | :--- |
| **EXIT** | 스크립트가 종료될 때 (정상/에러 무관) | 임시 파일 정리 등 청소용 |
| **INT** | `Ctrl+C` 입력 시 | 중단 처리 |
| **TERM** | `kill` 호출 시 | 안전한 종료 절차 수행 |
| **ERR** | 명령이 실패했을 때 (Bash 확장) | 디버깅 및 에러 로그용 |

## 4. 백그라운드 잡 동기화
| 변수 / 명령 | 설명 | 예시 |
| :---: | :--- | :--- |
| **`$!`** | 최근 실행된 백그라운드 잡의 PID | `pid=$!` |
| **`wait [PID]`** | 특정 PID 또는 모든 잡의 종료 대기 | `wait $pid` |
| **`timeout [T] [CMD]`** | 일정 시간만 실행하고 종료 (GNU) | `timeout 10s ./task.sh` |
