# 13. 모의고사 Q&A

시험 및 실무에서 자주 묻는 핵심 질문들과 그에 대한 명확한 답변을 정리합니다.

## 1. 핵심 단답형 Q&A
| 질문 (Question) | 답변 (Answer) |
| :--- | :--- |
| **`set -e`가 무시되는 곳은?** | `if/while` 조건식, `! cmd`, `&&/||`의 우변, 파이프라인(끝 제외) |
| **Bash 배열 길이는?** | `${#arr[@]}` |
| **`$@`와 `$*`의 차이는?** | `"$@"`는 각 인자 분리, `"$*"`는 하나의 문자열로 결합 (인용 시) |
| **배시 히어독 변수 방지는?** | 구분자를 인용 (`<<'EOF'`) |
| **종료 시 자동 실행 방법은?** | `trap '...' EXIT` |

## 2. 시나리오 해결 (Case Study)
| 상황 (Situation) | 해결책 (Solution) |
| :--- | :--- |
| **파일명에 공백이 섞여 있음** | `find ... -print0 \| xargs -0` 또는 `read -d ''` 사용 |
| **cron에서 실행이 안 됨** | 짧은 `PATH` 문제 → 절대경로 사용 또는 상단에 `PATH` 명시 |
| **서브셸 변수가 안 남음** | 리디렉션 `< file`을 사용하여 현재 셸의 `while`에서 처리 |
| **macOS sed 실패** | 인플레이스 옵션에 백업 인자 추가 (`sed -i '' ...`) |
| **필수 인자 누락 방지** | `${1:?usage message}` 패턴으로 즉시 종료 |

## 3. 명령 및 옵션 매칭
| 명령어 조합 | 의미 (Action) |
| :--- | :--- |
| `grep -nE 'A\|B'` | 줄 번호 포함, 확장 정규식으로 A 또는 B 검색 |
| `sed -n '1,10p'` | 1행부터 10행까지만 출력 |
| `awk -F, '{sum+=$3}'` | CSV 파일의 3번째 컬럼 합계 계산 |
| `sort -nr` | 데이터를 숫자로 인식하여 역순 정렬 |
| `read -r` | 백슬래시 이스케이프를 해석하지 않고 그대로 읽기 |

## 4. 실전 패턴 (Best Practice)
| 분야 | 모범 패턴 코드 |
| :--- | :--- |
| **안전한 실행** | `set -Eeuo pipefail` |
| **임시 작업** | `tmp=$(mktemp -d); trap 'rm -rf "$tmp"' EXIT` |
| **데이터 필터** | `sed '/^#/d; /^$/d'` (주석 및 빈 줄 제거) |
| **중복 카운트** | `sort \| uniq -c \| sort -nr` |
