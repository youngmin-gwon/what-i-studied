## 19. 초압축 회독 노트 (Quick Review)

시험 직전이나 빠른 개념 복습을 위해 핵심 내용을 챕터별로 요약합니다.

### 1. 챕터별 핵심 요약 (3-Line Summary)

| 챕터               | 핵심 내용 요약                                                               |
| :--------------- | :--------------------------------------------------------------------- |
| **01-03: 기초**    | 셔뱅, 로딩 순서(`/etc/profile` vs `~/.bashrc`), 변수/배열 및 인용 규칙(`"$VAR"`)      |
| **04-06: 제어/IO** | `[[ ]]` vs `[ ]` 차이, `while read` 루프, 리디렉션(`2>&1`) 및 파이프라인(`pipefail`) |
| **07-09: 툴킷/환경** | grep(검색), sed(편집), awk(필드), coreutils 활용 및 Cron/Systemd 환경 차이          |
| **10-12: 품질/보안** | `set -x` 디버깅, `shellcheck` 분석, `mktemp` 보안, GNU/BSD 포터빌리티 이슈           |
| **13-16: 실전/팁**  | 모의고사 Q&A, 빈출 스니펫, 랩 워크스루 및 트러블슈팅 Cookbook                              |

### 2. 잊기 쉬운 암기 포인트 (Critical Points)

| 포인트        | 내용                                                                           |
| :--------- | :--------------------------------------------------------------------------- |
| **종료 코드**  | 0(성공), 1(일반), 126(권한), 127(명령 없음), 128+N(시그널)                                |
| **특수 변수**  | `$#`(개수), `$@`(각각), `$*`(하나로), `$!` (막내 백그라운드 PID)                           |
| **패턴 제거**  | `${var#pat}`(앞 짧), `${var##pat}`(앞 길), `${var%pat}`(뒤 짧), `${var%%pat}`(뒤 길) |
| **시그널 번호** | 1(HUP), 2(INT, `^C`), 9(KILL), 15(TERM)                                      |

### 3. 빈출 실수 클리닉 (Common Mistakes)

| 현상                    | 올바른 해결 (Correct Way)                 |
| :-------------------- | :----------------------------------- |
| **공백 파일명 쪼개짐**        | `while read` 루프 또는 `xargs -0` 사용     |
| **Cron 에서 명령 실행 안 됨** | 절대경로 사용 또는 상단에 `PATH` 명시             |
| **macOS sed 치환 에러**   | `sed -i '' 's/a/b/' file` 처럼 빈 인자 추가 |
| **unset 변수로 종료**      | `${VAR:-default}` 처럼 기본값 패턴 사용       |
