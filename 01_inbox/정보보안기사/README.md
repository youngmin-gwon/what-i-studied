## 🎯 시험 정보

- **시험 형태**: 필기 + 실기
- **필기 시험**: 객관식 4지 선다형 100문항 (2시간 30분)
- **실기 시험**: 단답형 및 서술형 (2시간 30분)
- **합격 기준**: 60점 이상

## 📝 활용 방법

1. 각 챕터별로 순차적 학습
2. 핵심 키워드 암기
3. 실습 예제를 통한 이해도 확인
4. 모의고사를 통한 실전 대비
