## 🎯 학습 전략

### 시험 구성
- **1차 시험**: 리눅스 일반 (60문항, 60분)
- **2차 시험**: 리눅스 실무 (40문항, 120분)

### 효과적인 학습 방법
1. **순차적 학습**: 챕터 1부터 순서대로 학습하여 기초를 탄탄히 다지세요
2. **실습 병행**: 가상머신이나 실제 리눅스 환경에서 명령어를 직접 실행해보세요
3. **반복 학습**: 중요한 명령어와 개념은 여러 번 복습하세요
4. **문제 풀이**: 기출문제와 모의고사를 통해 실전 감각을 키우세요

### 중점 학습 영역
- **명령어 숙지**: 자주 사용되는 명령어의 옵션과 사용법
- **설정 파일**: 주요 설정 파일의 위치와 구조
- **시스템 관리**: systemd, 네트워크, 패키지 관리
- **문제 해결**: 로그 분석, 트러블슈팅 기법

## 💡 학습 팁

- 각 챕터는 독립적으로 구성되어 있지만, 상호 참조를 통해 연관 개념을 쉽게 찾을 수 있습니다
- 코드 블록의 예제는 실제 환경에서 테스트해보세요
- 이해가 안 되는 부분은 표시해두고 나중에 다시 학습하세요
- 시험 전 마지막 주에는 전체 내용을 빠르게 복습하세요
