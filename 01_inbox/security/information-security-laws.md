## 🌐 개요 (Overview)

국내 **정보보호 및 개인정보보호** 관련 주요 법률을 다룹니다. 정보보안기사 시험에서 자주 출제되는 핵심 조항을 정리합니다.

---

## 📋 1. 정보통신망법

**정보통신망 이용촉진 및 정보보호 등에 관한 법률**

### 목적

정보통신망 이용 촉진, 이용자 **개인정보 보호**, 건전하고 안전한 정보통신환경 조성

### 주요 용어

| 용어 | 정의 |
|------|------|
| **정보통신망** | 전기통신설비 + 컴퓨터 연결한 정보통신체계 |
| **정보통신서비스 제공자** | 전기통신사업자 + 영리 목적 정보 제공자 |
| **침해사고** | 해킹, 바이러스, DoS 등으로 발생한 사태 |

### CISO (정보보호최고책임자) - 법 제45조의3

| 항목 | 내용 |
|------|------|
| **대상** | 정보통신서비스 제공자 |
| **직급** | **임원급** |
| **신고** | 과기정통부장관에게 신고 |
| **겸직 금지** | 자산총액 **5조원 이상** 기업 |

**CISO 업무**:
- 정보보호 관리체계 수립/운영
- 취약점 분석/평가
- 침해사고 예방/대응
- 보안성 검토, 암호화 검토

### ISMS 인증 의무 - 법 제47조

| 대상 | 기준 |
|------|------|
| **ISP** | 정보통신망서비스 제공자 |
| **IDC** | 집적정보통신시설 사업자 |
| **매출액** | 정보통신서비스 **100억 원** 이상 |
| **이용자 수** | 일평균 **100만 명** 이상 |
| **상급종합병원** | 매출/세입 **1,500억 원** 이상 |

**인증 유효기간**: 3년 (매년 사후 심사)

### 개인정보 보호 (특례)

| 항목 | 내용 |
|------|------|
| **유효기간제** | 1년 미이용 시 파기 또는 분리 보관 (30일 전 통지) |
| **유출 통지** | **24시간 이내** 이용자 통지 + 방통위/KISA 신고 |
| **징벌적 손해배상** | 손해액의 **3배** 이내 |
| **법정 손해배상** | **300만 원** 이하 |

### 스팸 규제

- **Opt-in 방식**: 수신자의 **사전 동의** 필요
- 동의 없이 광고 전송 금지

## 📋 2. 개인정보 보호법

**일반법** 성격으로, 공공/민간 모든 개인정보처리자에 적용됩니다.

### 주요 정의 - 법 제2조

| 용어 | 정의 |
|------|------|
| **개인정보** | 살아 있는 개인을 알아볼 수 있는 정보 (결합 정보 포함) |
| **처리** | 수집~파기까지 **모든 행위** |
| **개인정보처리자** | 업무 목적으로 개인정보 처리하는 자 |
| **가명정보** | 추가 정보 없이는 개인 식별 불가한 정보 |

### 개인정보 보호 원칙

```plaintext
✅ 목적의 명확성
✅ 최소 수집
✅ 목적 외 이용 금지
✅ 정확성/최신성 보장
✅ 안전한 관리
✅ 사생활 침해 최소화
✅ 익명/가명 처리 우선
```

### 수집 및 이용 - 법 제15조

| 허용 사유 |
|----------|
| 1. **정보주체 동의** |
| 2. 법률 규정 / 법령상 의무 |
| 3. 공공기관 소관 업무 |
| 4. **계약 체결/이행** 불가피 |
| 5. 생명/신체/재산 이익 보호 |
| 6. **정당한 이익** 달성 필요 |

### 안전조치의무 - 법 제29조

| 조치 | 내용 |
|------|------|
| 내부 관리계획 | 수립 |
| 접속기록 | 보관 |
| 암호화 | 비밀번호, 고유식별정보 |
| 보안프로그램 | 설치/운영 |
| 물리적 접근 통제 | |

### 유출 통지 - 법 제34조

| 항목 | 기준 |
|------|------|
| **통지 시한** | **5일 이내** |
| **공고 조건** | **1,000명** 이상 유출 시 |
| **신고 대상** | 보호위원회/KISA |

### 가명정보 특례 - 법 제28조의2 (데이터 3법)

| 항목 | 허용 조건 |
|------|----------|
| **동의 없이 처리 가능** | 통계, 과학적 연구, 공익적 기록 보존 |
| **금지** | 특정 개인 식별 목적 처리 |

## 📋 3. 정보통신기반 보호법

**주요정보통신기반시설**을 침해로부터 보호합니다.

### 추진 체계

| 기관 | 역할 |
|------|------|
| **정보통신기반보호위원회** | 국무총리 소속, 기본계획 심의 |
| **중앙행정기관장** | 기반시설 지정 |
| **관리기관장** | 취약점 분석/평가, 보호대책 수립 |

### 주요정보통신기반시설 지정 - 법 제8조

| 대상 분야 |
|----------|
| 국가안보, 행정, 금융, 통신, 운송 등 |
| 파괴 시 영향이 큰 전자적 제어/관리 시스템 |

### 취약점 분석·평가 - 법 제9조

- 관리기관장이 **정기적** 수행
- 전담반 구성 또는 전문기관 위탁 가능

### ISAC (정보공유·분석센터)

- 분야별 취약점/침해요인 정보 공유
- 금융 ISAC, 통신 ISAC 등

## 📋 4. 위치정보법

**위치정보의 보호 및 이용 등에 관한 법률**

### 정의

이동성 있는 물건이나 개인이 특정 시간에 존재하거나 존재했던 **장소에 관한 정보**

### 긴급구조 - 법 제29조

| 요청 기관 | 거부 가부 |
|----------|:--------:|
| 긴급구조기관 (소방, 해경) | **거부 불가** |
| 경찰관서 | **거부 불가** |

## 📋 5. 신용정보법

**신용정보의 이용 및 보호에 관한 법률**

### 마이데이터 (본인신용정보관리업)

| 항목 | 내용 |
|------|------|
| **전송요구권** | 본인 신용정보를 마이데이터 사업자에게 전송 요구 |
| **프로파일링 대응권** | 자동화 평가 결과 설명/이의 제기 권리 |

## 📋 6. 클라우드컴퓨팅법

**클라우드컴퓨팅 발전 및 이용자 보호에 관한 법률**

### CSAP (클라우드 보안 인증)

| 항목 | 내용 |
|------|------|
| **인증 기관** | 과기정통부 |
| **공공기관** | 인증 서비스 우선 이용 |

### 이용자 정보 보호

| 금지 사항 |
|----------|
| 동의 없이 제3자 제공 |
| 목적 외 이용 |
| 서비스 종료 시 미파기/미반환 |

## 📊 주요 법률 비교

| 법률 | 대상 | 핵심 |
|------|------|------|
| **정보통신망법** | 정보통신서비스 제공자 | CISO, ISMS, 유출 24시간 |
| **개인정보보호법** | 개인정보처리자(전체) | 수집~파기, 유출 5일 |
| **정보통신기반보호법** | 기반시설 관리기관 | 취약점 분석/평가 |
| **위치정보법** | 위치정보사업자 | 긴급구조 협조 |
| **신용정보법** | 금융권 | 마이데이터, 전송요구권 |
| **클라우드법** | 클라우드 제공자 | CSAP 인증 |
