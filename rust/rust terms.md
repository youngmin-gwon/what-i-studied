## macro

! 가 들어가는 function 모양

```rust
println!();
```

추후에 추가할테니 현재로서는 !가 들어가면 function 이 아니라 macro 가 불린다는 것만 이해

## Prelude

a set of items defined in the standard library that it brings into the scope of every program
